#PawPrints Project

This project was completed for the class Interface Design Methodology and Rapid Prototyping, which covered the basics of Android Application development and various user interface principles.

These folders contain the .apk and source code of the finished project, PawPrints, which is a proof-of-concept of an app which can crowdsource the discovery of lost pets.

This projected was completed as part of a group assignment over three main stages, the final stage being the finished application.

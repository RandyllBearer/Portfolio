#Introduction to Systems Software

This class covered the C programming language, basics of computer hardware/bit manipulation/etc. Some projects from this class could not be uploaded due to University policies.

Project 1: A black-jack game. Covered the basics of the C language.

Project 2: A .jpeg metadata parser. Covered input/output, byte processing.

Project 4: A program which reads from an input file byte-by-byte, adds each byte to a node in a tree structure. At the end of the program, prints out all byte sequences which comprise more than 4 nodes. Covers tree structure/traversal/more i/o

Project 5: A TelNet listener server. Covered the basic of tcp/ip protocols, pthreading, mutex-locks, server-client architecture, sockets, etc.
